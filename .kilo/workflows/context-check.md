# Context Check

Перед началом работы выполни поиск релевантного контекста в OpenViking.

## Шаги

1. Проверь соединение с OpenViking через `health`
2. Выполни семантический поиск: `find "{{query}}"` с запросом, релевантным текущей задаче
3. Если найдены результаты с score > 0.5:
   - Прочитай `.overview` наиболее релевантных результатов
   - Верни summary найденного контекста
4. Если ничего не найдено — сообщи, что контекст отсутствует

Используй `$ARGUMENTS` как поисковый запрос.
